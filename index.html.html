Unit 1 — History & Fundamentals of Computers

From the abacus to modern CPUs — the full story

📅Timeline of Computing
~3000 BC
Abacus — the first device known to perform calculations using sliding beads.
1642
Pascaline — Blaise Pascal developed the first mechanical calculator.
1801
Punched Cards — Joseph Jacquard introduced punched cards to control weaving machines.
1820s
Difference Engine & Analytical Engine — Charles Babbage designed both; considered father of computers.
1840s
Ada Lovelace — wrote algorithms for Babbage's engine; considered the first computer programmer.
1890
Herman Hollerith — developed a machine to process census data; company eventually became IBM.
1946
ENIAC — one of the first fully electronic computers, used vacuum tubes.
1950s
Von Neumann Architecture — John von Neumann introduced storing both data and instructions in memory.
1950s
Transistors — replaced vacuum tubes; made computers smaller, faster, and more reliable.
1950s
FORTRAN & COBOL — first high-level programming languages emerged.

🖥️Basic Computer Operations
Every computer performs four basic operations:

1. Input — Receiving data from input devices (keyboard, mouse).
2. Processing — Performing arithmetic and logical operations (CPU).
3. Output — Displaying/printing results (monitor, printer).
4. Storage — Saving data for later use (RAM, HDD, flash drive).
🧠Central Processing Unit (CPU)
The CPU is the brain of the computer. Its primary function is to perform arithmetic and logical operations. It fetches instructions from RAM, decodes them, executes them, and stores results.
The CPU cannot work directly on data stored in Secondary Storage — data must first be loaded into RAM (Main Memory).
💾Memory Types
⚡
Main Memory (RAM)
Temporary, fast storage. Holds programs/data during execution. Each cell has a unique address. Lost when power is off.
💿
Secondary Storage
Permanent storage. Flash drives, CDs, DVDs, hard disks. Data is kept when power is off.
⌨️Input & Output Devices
Input Devices: Keyboard, Mouse — feed data into the computer.
Output Devices: Monitor, Printer — display or produce results from the computer.
💿System Programs vs Application Programs
System Programs (System Software): Control and manage the computer's hardware and basic operations. The most important system program is the Operating System (OS).
The Operating System is the first program to load when you turn on your PC. It controls the computer, manages all system activities, and runs application programs.
Application Programs: Programs designed for specific tasks that the user wants to perform. Examples: Word Processor, Spreadsheet, Web Browser, Games.
🔧Role of the Operating System
The OS acts as the bridge between the user, application programs, and hardware. Responsibilities include:

• Managing memory (RAM allocation)
• Managing input/output devices
• Running and scheduling application programs
• Handling file storage on secondary storage
• Providing the user interface (desktop, taskbar, etc.)
🔢Binary Numbers
A binary number is a number composed of only 0s and 1s.

Computers use binary because their circuits have two states: ON (1) and OFF (0).

All data inside a computer — text, images, numbers, sound — is stored as binary digits (bits).
Bit = 1 binary digit (0 or 1)
Byte = 8 bits
Kilobyte (KB) = 1024 bytes
Megabyte (MB) = 1024 KB
BitByteBinary 0s and 1sON/OFF statesMemory address

Unit 2 — Introduction to C++

Syntax, data types, variables, operators, and strings

🏗️C++ Program Structure
Every C++ program starts with:
#include <iostream> — includes the input/output library
using namespace std; — allows writing cout instead of std::cout

Every program must have a main() function — this is where execution begins.
return 0; at the end tells the OS the program finished successfully.
cout << "Hello World!"; — outputs text to the screen
\n — escape sequence to move cursor to a new line
endl — manipulator that also moves to a new line (and flushes the buffer)
Both \n and endl achieve the same visual result.
💬Comments in C++
// This is a single-line comment

Comments are ignored by the compiler. They are used to explain code to humans.
🔢
int
Stores whole numbers. Example: int x = 5;
🔣
double
Stores decimal/fractional numbers. Example: double x = 2.8;
🔤
string
Stores text. Uses double quotes. Example: string s = "Hello";
🔡
char
Stores a single character. Uses single quotes. Example: char g = 'A';
✅
bool
Stores true or false only. Occupies 1 byte in memory. Example: bool b = true;
💾
Memory Size
bool: 1 byte | int: 4 bytes | double: 8 bytes | char: 1 byte
➕Arithmetic Operators
+ Addition  |  - Subtraction  |  * Multiplication
/ Division  |  % Modulus (remainder after integer division)

Order of operations (BODMAS/PEMDAS) applies. Example: 10 * 2 + 5 = 25
🔁Increment & Decrement
++z — increases value by 1 (prefix increment)
--z — decreases value by 1 (prefix decrement)

Example: if z = 5, then ++z makes z = 6, then --z makes z = 5 again.
⚖️Comparison & Reference Operators
== — compare two values (equal to)
& — reference operator (creates a reference variable to an existing variable)
* — pointer operator
💬String Functions
.length() — returns the number of characters in a string
.size() — alias of .length() (both do the same thing)

The dot (.) operator accesses functions that belong to the string object.
The empty parentheses () indicate it is a function/method being called.

Note: Spaces count as characters! "A B C" has length 5, not 3.
🔍String Indexing
Strings are zero-indexed — first character is at index 0.
Example: string s = "Hello";
s[0] = H  |  s[1] = e  |  s[4] = o (last character)

Each element accessed by s[i] is of type char, not string.
You can change a character: s[0] = 'J'; → s becomes "Jello"
📋Arrays in C++
An array stores multiple values of the same type.
Declared with square brackets: int arr[5];

Array indexes start at 0 (zero-indexed).
To access elements: arr[0], arr[1], etc.
🏛️Classes & Objects
class keyword is used to create a class in C++.
To create an object: MyClass myObj;

C++ supports inheritance — a class can inherit attributes and methods from another class. (True)
🔧Useful Functions
max(x, y) — returns the highest value between x and y
functionName() — how to define and call a function
return — keyword used to return a value from a function
break — stops a loop or exits a switch case
void — function return type when no value is returned

Unit 3 — Control Flow & Loops in C++

If statements, switch, while loops, for loops, and nested structures

🔀If / Else If / Else
Syntax: if (condition) { ... } else if (condition2) { ... } else { ... }

Nested If: An if statement inside another if statement.
Used when conditions depend on each other (e.g., check attendance first, then grade).
Starting an if statement: if (x > y) — no semicolon after the condition!
The condition must be in parentheses.
💡Nested If Example — Student Grades
Check attendance (≥70%) first. If met, check exam score:
• Score 90–100 → Grade A
• Score 80–89 → Grade B
• Score 70–79 → Grade C
• Score <70 → Fails grade requirement
If attendance <70% → "Failed the attendance requirement"
🔃Switch Statement
Used when a variable can take one of several specific values.
Syntax: switch (variable) { case 1: ...; break; default: ...; }

break is required to prevent fall-through to the next case.
default handles any value not matching the listed cases.
📐Area Calculator Example
Switch on shape choice (1–4):
case 1: Triangle → 0.5 * base * height
case 2: Rectangle → length * width
case 3: Square → side * side
case 4: Circle → 3.14159 * r * r
default: Invalid selection
🔁While Loop
Syntax: while (condition) { ... }
Repeats as long as the condition is true.

Starting a while loop: while (x > y) — no semicolon.
🔄Do-While Loop
Syntax: do { ... } while (condition);
Executes the body at least once before checking the condition.

Example — Passcode: Ask for a passcode repeatedly until the correct one is entered.
🔢For Loop & Nested For Loops
Syntax: for (int i = 0; i < n; i++) { ... }

Nested For Loops: A for loop inside another for loop.
Example — Multiplication tables: outer loop selects the table number, inner loop iterates multipliers from 1 up to the table number itself.
🍽️Cafeteria Menu (Switch Example)
Display a menu with 5 items. Prompt user for choice 1–5. Use switch to show price. Default case shows "Invalid choice".
switch (choice) {
  case 1: cout << "$2"; break;
  case 2: cout << "$1"; break;
  ...
  default: cout << "Invalid";
}
🔐Passcode (Do-While Example)
Ask user to enter passcode. If wrong, print "Incorrect! Try again." Keep looping until the correct code (1234) is entered. Then print "Access Granted!".
✖️Multiplication Tables (Nested For Example)
Ask user to enter n. Print tables for n and n+1. Each table goes from i * 1 up to i * i (stops at the number itself).
